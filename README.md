# SENG2050_Lab2
This repository consists of exercises for Lab 2 of SENG2050 course

The lab focuses on building a basic **Java web application** using **Servlets** and **Apache Tomcat**, introducing server-side request handling and form submission.

## Features
- Embedded Tomcat server setup using Java
- Multiple servlets including:
  - `HelloWorldServlet`
  - `SimpleMessage`
  - `GenerateValidHTML`
  - `GreetingServlet`
- Dynamic HTML generation from servlets
- Reading request parameters from URLs and form submissions
- HTML helper class (`HtmlGen`) to improve code maintainability
- HTML form for capturing user input
- Support for both **GET** and **POST** requests
- Client-side form validation using JavaScript

## Technologies Used
- Java
- Jakarta Servlets
- Apache Tomcat (embedded)
- HTML
- JavaScript
- Maven

## How to Run
1. Open the project in **VS Code**
2. Navigate to the `lab2` directory
3. Run `Main.java` to start the embedded Tomcat server
4. Access the application in a browser at:
http://localhost:8080
